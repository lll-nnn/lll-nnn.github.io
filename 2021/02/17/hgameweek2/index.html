lazy.php: 

<?php
$filter = ["SESSION", "SEVER", "COOKIE", "GLOBALS"];

// 直接注册所有变量，这样我就能少打字力，芜湖~

foreach(array('_GET','_POST') as $_request){
    foreach ($$_request as $_k => $_v){
        foreach ($filter as $youBadBad){
            $_k = str_replace($youBadBad, '', $_k);
        }
        ${$_k} = $_v;
    }
}

第一个循环，将GET和POST得到的数据给$_request
第二个循环，将键值分别赋给$_k和$_v
第三个循环，过滤$filter中的词
传入username=admin
经过${$_k} = $_v,就成了$username=admin
给的wp中是_SESSISESSIONON[username]=admin
经过lazy.php后是$_SESSION=$_v而$_v是一个数组，array(1) { ["username"]=> string(5) "admin" }
所以$_SESSION[username]=admin达到变量覆盖
