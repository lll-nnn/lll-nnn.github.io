页面一个表情，查看源代码

访问source.php,得到PHP代码 

     <?php
        highlight_file(__FILE__);
        class emmm
        {
            public static function checkFile(&$page)
                {
                $whitelist = ["source"=>"source.php","hint"=>"hint.php"];     //$whitelist有"source.php"和"hint.php"两个字符串
                if (! isset($page) || !is_string($page)) {       //要求$page存在且是字符串
                    echo "you can't see it";

                    return true;

                echo "you can't see it";
                return false;

  • 分析

首先，看到个hint.php,传个file=hint.php得到的结果是：flag not here, and flag in ffffllllaaaagggg这个告诉了我们flag所在的文件名；

分析一下为什么：第一个、第二个if可以过去，截取函数那里，先加个 ？ 然后截取第一个字符到 ？ ，还是hint.php;没有经过url编码，解码后还是那样；下面一个也一样；最后include函数包含并运行hint.php，得到其中的内容。

接下来是include函数的利用：

  • 解决

传参：file=hint.php?/../../../../../../ffffllllaaaagggg
得到flag；其中file=hint.php?/将其当作文件夹，../到上一级目录，最少四个，因为一般Web服务的文件夹在/var/www/html目录中，再加上source.php?/这个“文件夹”，所以我们总共需要../四次来返回到根目录,从而打开文件，得到flag。

  • 知识点
    • in_array函数：检查数组中是否存在某个值
      语法：````in_arry(mixed $needle,arry $haystack,bool $strict =false):bool ``
      如果没有设置$strick=true,将比较类型是否相同
      mixed的意思是可以为任意类型
      如果needle是字符串，则比较是区分大小写的
      如果找到needle则返回true，否则返回false
      MORE
    • mb_substr函数：返回字符串的一部分，substr()函数只针对英文字符，mb_substr还支持中文文字
      语法：mb_substr ( string $str , int $start , int $length = NULL , string $encoding = mb_internal_encoding() ) : string
      $str:提取目标
      $start:开始提取的位置；负数的话从尾处第start个字符开始
      $length ：提取长度；如果省略或传入NULL，则会提取到尾部
      $encoding: 为字符编码；如果省略或是NULL，则使用内部字符编码。
      返回值：提取到的字符
      MORE
    • is_string函数：检测变量是否是字符串。
      MORE
    • mb_strpos函数：查找字符串在另一个字符串中首次出现的位置
      语法：mb_strpos ( string $haystack , string $needle , int $offset = 0 , string $encoding = mb_internal_encoding() ) : int
      第一个字符的位置是 0，第二个字符的位置是 1，以此类推。
      $offset: 搜索位置的偏移。如果没有提供该参数，将会使用 0。负数的 offset 会从字符串尾部开始统计。
      MORE
