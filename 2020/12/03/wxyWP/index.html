伪协议1

题目：

                    针不戳，吃柠檬的感觉针不戳
    <?php
    error_reporting(0);
    echo '<center><strong>针不戳，吃柠檬的感觉针不戳</strong></center>';
    highlight_file(__FILE__);
    $file=$_GET['file'];
    if (isset($file)){
        if(preg_match('/php|file|zip|bzip|zlib|base|data/',$file)){
            echo ('hacker!!!');
        }else {
            include $file;          ////include 语句包含并运行指定文件。

这次的题目值得注意的是这个preg_match函数中没有 i，也就是说这个正则匹配是区分大小写的，所以将涉及到的过滤的词换成大写即可。

伪协议2

题目：

                针不戳，吃柠檬的感觉针不戳
    <?php
    error_reporting(0);
    echo '<center><strong>针不戳，吃柠檬的感觉针不戳</strong></center>';
    highlight_file(__FILE__);
    $file=$_GET['file'];
    if (isset($file)){
        if(preg_match('/php|file|zip|bzip|zlib|base|data/i',$file)){
            echo ('hacker!!!');
        }else {
            $a=str_replace("troye","",$file);       ////str_replace替换函数，意思是将$file中的troye替换没有。
            include $a;
        }
    }
    #不对劲，总感觉写错了啥，算了,继续吃柠檬~
    ?>

这一次preg_match函数中加上了i，区分大小写了，但下面的str_replace函数提供了另一种绕过方式，所以可以将preg_match过滤的词中插入troye便可绕过输入?file=ptroyehp://filter/read=convert.btroyease64-encode/resource=flag得到base64编码：
解码后得到flag：

知识点总结

include函数：包含并对给定的文件进行计算。
str_replace函数：替换字符串中的一些字符（区分大小写）。而 str_ireplace() 函数执行不区分大小写的搜索。
CTF伪协议常用的两句话：
php://filter/read=convert.base64-encode/resource=[文件名]读取文件源码
php://input+[POST DATA] 执行PHP代码
